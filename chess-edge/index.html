Improvements of interface

  • We created a minimalistic and modern interface.
  • We placed arrows next to items so the user knows that there is something there and can explore it.
  • We placed subtopics in a box to the right of the page.
  • We added a navigation footer with arrows to the left and right and the title of the subject.
  • We used headroom.js so that the header and the footer get out of the way when you don't need them.
  • We made it all responsive.

Making the system adaptive

  • We added quizes to test the user's knowledge.
  • Fragments of a topic that relate to another topic are only shown when both topics have been studied. For example, castling appears only when both the rook and the king have been studied.
  • We recommend topics based on previous knowledge.

End summary

  • Responsive, minimalistic and modern interface
  • Dropdown arrows
  • Subtopics in a box to the right
  • Navigation footer with arrows plus subject title
