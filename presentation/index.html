What is FiestApp?

  • Wish your friends a happy holiday.
  • Create your own holidays.
  • Find fun local events.

But why?

  • To connect with your friends.
  • To start a conversation.
  • To do something new and exciting.
  • To start a tradition.
  • To never be bored again.

Requirements

  • Connectivity
    • Fetching holidays, measuring popularity, twitter trends
  • User Interaction
    • Main app interface
  • Use of sensors
    • GPS for local events, shake for random holiday, camera to add photos
  • Android services and activities
    • Sharing, map, view, push notifications, Google Wallet

Feasibility

  • Start with basic application.
  • Adding more functionality per iteration.
  • Version control.
